# Go-To-Market (GTM) Playbooks

These playbooks define the acquisition and activation strategies for the four primary segments of the Indikin ecosystem.

## Strategy Manifest

| Playbook | Target Segment | Key Mechanism |
|----------|----------------|---------------|
| [GTM_Filmmakers.md](./Indikin_GTM_Filmmakers.md) | Independent Creators | Personal outreach + ONBA automation. |
| [GTM_Fans.md](./Indikin_GTM_Fans.md) | Core Supporters | Star Pass tiers + filmmaker token bonding. |
| [GTM_Streaming_Consumers.md](./Indikin_GTM_Streaming_Consumers.md) | General Viewers | IndiFlix subscription + PPV models. |
| [GTM_Liquidity_Providers.md](./Indikin_GTM_Liquidity_Providers.md) | DeFi Investors | PulseChain -> Ethereum index yield. |
| [TGE_Strategy.md](./INDIKIN_TGE_GTM_Strategy.md) | Market Entry | Token Generation Event (TGE) execution. |
