# Scenario Model (AI Budget)

## Scenario definitions
- Base case: expected adoption + expected unit costs
- Upside case: higher adoption + optimized unit economics
- Downside case: slower adoption + higher unit costs

## Inputs to vary
- Active users / workload volume
- Unit API/model costs
- Infrastructure utilization
- Staffing pace

## Output metrics
- Monthly burn (AI-only)
- 12-month AI budget
- Cost per active filmmaker
- Cost per AI-generated output
- Runway impact under each scenario
